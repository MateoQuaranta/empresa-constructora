# assets/ — Empresa Constructora

## Estructura esperada

Copiá tus imágenes existentes en las rutas indicadas en portfolio.json.

### Imágenes requeridas

| Ruta | Descripción |
|------|-------------|
| `assets/hero-bg.jpg` | Imagen de fondo del hero (mínimo 1920×1080px, JPG optimizado) |
| `assets/nosotros-principal.jpg` | Foto del equipo o una obra destacada (800×520px) |
| `assets/og-image.jpg` | Imagen para compartir en redes sociales (1200×630px) |
| `assets/favicon.png` | Favicon del sitio (32×32px o 64×64px) |
| `assets/portfolio/` | Imágenes del portfolio (ver portfolio.json para los nombres exactos) |
| `assets/portfolio/placeholder.jpg` | Imagen fallback si una foto no carga (400×300px) |

### Recomendaciones

- Formato: JPG para fotos, PNG para logos/íconos
- Portfolio cards: 800×600px o relación 4:3 (se cortan con object-fit: cover)
- Optimizá antes de subir: usá https://squoosh.app o similar
- Si usás imágenes del sitio Wix original, descargalas desde el panel o usando las URLs del navegador
